## MongoDB Commands :------------------------------------

###### Start your MongoDB database

`"C:\Program Files\MongoDB\Server\4.0\bin\mongod.exe" --dbpath="c:\data\db"`

###### Connect to MongoDB

`"C:\Program Files\MongoDB\Server\4.0\bin\mongo.exe"`

###### Start the MongoDB service.

`net start MongoDB`

###### Stop the MongoDB service.

`net stop MongoDB`
